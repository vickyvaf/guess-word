## Getting Started

1.  **Clone the repository**

    ```bash
    git clone <repository-url>
    cd guess-word
    ```

2.  **Install dependencies**

    ```bash
    bun install
    ```

3.  **Run the development server**

    ```bash
    bun run dev
    ```

4.  **Build for production**
    ```bash
    bun run build
    ```
